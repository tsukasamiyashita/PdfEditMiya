# PdfEditMiya v1.8.0

PdfEditMiyaは、PDFファイルの操作とデータ変換を一括で行う多機能ツールです。

## 新機能と改善 (v1.8.0)

* **「Chain of Thought (思考の連鎖)」による列ズレの完全排除:**
    * AIがいきなりデータを抽出するのではなく、「この表は何列あるか」「各列にはどんなデータが入るか」を事前に自己分析（自己定義）するアーキテクチャを導入しました。これにより、AIが空白セルを読み飛ばして列がずれてしまうエラーをシステム的に防ぎます。
* **マスターヘッダーによる縦列の強制同期（マッピング機能）:**
    * 集約データを作成する際、1ページ目で分析・構築した「マスターのカラム構造（列キー）」を記憶し、2ページ目以降のすべてのデータをそのマスターキーに沿って強制的に整列・マッピングさせる仕組みを追加しました。ページごとのブレを吸収し、綺麗なデータベースができあがります。
* **プロフェッショナルなモダンUIへの進化:**
    * 前回のアップデートで導入したカード風デザインをベースに、カラーパレット（ブルーとグレーの階調）、余白、フォントサイズを細かく再調整し、より洗練された見栄えにアップグレードしました。

## 機能一覧

* **結合:** 複数PDFを1つにマージ
* **分割:** 1ファイルをページごとに分割
* **回転:** 全ページを指定角度で回転
* **Text抽出:** PDF内部のテキストデータを抜き出し
* **AI抽出:** 手書き文字やスキャン画像をAI(Gemini/Tesseract)で高精度に読み取り、指定形式へ出力
    * カラムの自動認識による不要データの排除機能を搭載
    * 集約データへの「元ファイル名」列の自動付与（トレーサビリティの確保）
* **Excel変換:** 活字の表データをExcelへ
* **画像変換:** JPEG / PNG へ変換
* **DXF変換:** CADデータへ変換 (高精度ハイブリッド版)
