PdfEditMiya v2.3.0

PdfEditMiyaは、PDFファイルの編集（結合・分割・回転）から、AIを用いた高度なデータ抽出、そして抽出したデータの自動集約までをワンストップで行うWindows向けデスクトップアプリケーションです。

🌟 主な特徴

直感的な操作: ファイルやフォルダを選択し、処理を選んで実行するだけの簡単設計。

高精度AIデータ抽出: Google Gemini APIを利用し、かすれた手書き文字や複雑な表の罫線を高精度に認識。無料枠でもバーストエラーを自動回避する機能を搭載。

高度なAPI詳細設定: 以下の設定を「無料枠」と「課金枠」それぞれで独立して設定・保存できます。

使用モデル: 目的に合わせた Gemini モデルの選択

RPM制限: バーストエラーを防ぐ1分間のリクエスト数制限

同時処理スレッド数: 安全な「直列処理」から超高速な「並列処理」までの調整

Temperature: 抽出データの正確性を高めるパラメータ

安全フィルタ無効化: 業務文書等でのエラー（誤検知）を回避

最大トークン数: 情報量が多いPDFでのデータ途切れ防止

カスタムプロンプト（箇条書き対応）: 複数行にわたる独自のルール（金額のカンマ削除、日付形式など）を入力でき、ワンクリックでお気に入りとして保存・復元が可能です。

ローカルOCR対応: オフライン環境でも使えるTesseract OCRに対応。

多彩な出力フォーマット: Excel (.xlsx, .xlsm, .xls)、CSV、テキスト、JSON、Markdown、Word (.docx)、各種画像フォーマット出力に対応。

ピンポイント抽出（クロップ機能）: プレビュー画面からドラッグで指定した範囲（複数可）のみを超高速で抽出可能。

スマートデータ集約: バラバラに抽出された複数ページの表データを、列名を自動認識して1つのマスターファイルに結合・集約（AIによる高度な意味的統合にも対応）。

設定の保存機能: APIキーや出力形式など、現在の選択状態を保存し次回起動時に復元可能。

🚀 基本的な使い方

アプリの起動: ダウンロードした PdfEditMiya-v2.3.0.exe をダブルクリックして起動します。

対象の選択: 画面上部の「📄 ファイルを選択」または「📁 フォルダを選択」から処理したいPDFやデータを選びます。

設定の調整:

保存先: 同じフォルダ、または任意のフォルダを指定。

抽出設定: 使用するエンジン（標準 / Tesseract / Gemini）と出力形式を選択。

範囲指定（任意）: 特定の表や箇所だけを抽出したい場合は「抽出範囲を選択」から領域を指定します。

実行: 画面下部のボタン（「🚀 選択した抽出・変換を実行」や「結合」など）を押して処理を開始します。

⚠️ 【重要】初回起動時の注意点（青い警告画面について）

本アプリ（.exeファイル）を初めて実行した際、青い画面で**「Windows によって PC が保護されました」**という警告（SmartScreen）が表示される場合があります。
これはインターネットからダウンロードした新しいアプリに対して表示されるWindowsの標準的な保護機能であり、ウイルスではありませんのでご安心ください。

【回避して起動する手順】

警告画面の左側にある 「詳細情報」 という文字をクリックします。

右下に 「実行」 ボタンが新しく表示されるので、それをクリックするとアプリが立ち上がります。

※ 初回起動時は、裏側でアプリを動かすための準備処理が行われるため、画面が表示されるまでに十数秒〜数十秒ほど時間がかかる場合があります。ボタンを押した後はそのままお待ちください。

⚙️ AI機能を使うための事前準備

本アプリの強力な「AIデータ抽出」機能をフル活用するためには、以下の簡単な準備が必要です。

Gemini API を使う場合（推奨・超高精度）

最新のAIモデルを利用して高精度なテキスト・表データの抽出を行います。インターネット接続が必要です。

ブラウザで Google AI Studio にアクセスし、Googleアカウントでログインします。

画面左上の「Create API key」を押し、新しいプロジェクトでAPIキーを作成します。

発行された長い文字列（APIキー）をコピーし、本アプリのメイン画面から「⚙️ 詳細設定 (APIキー / モデル / 制限)」ボタンを押します。

設定したいプラン（無料枠 または 課金枠）のタブを開き、その中の「APIキー」欄に右クリックで貼り付けます。

「テスト」ボタンを押し、通信が成功することを確認してください。

※使用するモデルや、その他の高度な抽出設定（スレッド数やカスタムプロンプトなど）も、各タブ（無料/課金）ごとに別々に設定できます。

カスタムプロンプト欄では、改行を含む箇条書きで指示を入力でき、「保存」ボタンを押すことでいつでもリストから復元可能になります。

画面下部の**「設定を適用して閉じる」**ボタンを押して設定を完了します。

※右上の「×」ボタンや「キャンセル」ボタンを押した場合は変更が破棄されます。

最後に、画面上部の「実行プランの選択」で、実際にデータ抽出で使用するプランを選んでください。

入力した設定は、メイン画面の「💾 現在の選択項目を保存」ボタンを押すことで保存され、次回から入力不要になります。

Tesseract（ローカルOCR）を使う場合のご準備（オフライン環境向け）

インターネットに接続できない環境でも、無料の文字認識（OCR）機能をご利用いただけます。ご利用には、お使いのパソコンに「Tesseract（テッセラクト）」という専用ソフトを追加していただく必要がございます。以下の手順に沿ってご準備をお願いいたします。

ダウンロードページを開く
こちらの Tesseractインストーラー配布ページ（外部サイト） をクリックして開きます。

インストーラーのダウンロード
ページ内にある「tesseract-ocr-w64-setup-v5.3.3.20231005.exe」のような名前のリンク（※バージョンを示す数字は時期により異なります）をクリックし、ファイルをパソコンにダウンロードします。

インストールを開始する
ダウンロードしたファイルをダブルクリックして開きます。画面の案内に従って「Next（次へ）」を進めてください。

【重要】日本語データの追加
途中で「Choose Components（追加機能の選択）」という画面が表示されます。
一覧の中にある Additional language data (download) の左側にある「+」マークを押して展開し、「Japanese」 と 「Japanese (vertical)」 の2箇所に必ずチェックを入れてください。（これを行わないと日本語が正しく読み取れません）

インストールの完了
「Destination Folder（インストール先）」の画面では、絶対に場所を変更せず（初期設定の C:\Program Files\Tesseract-OCR のまま）、「Next」を押して最後まで完了させてください。

以上の手順で準備は完了です。アプリ内の設定で「Tesseract (ローカルOCR)」をお選びいただけるようになります。
