# PdfEditMiya v1.3.0

PdfEditMiyaは、PDFファイルの操作とデータ変換を一括で行う多機能ツールです。

## 新機能 (v1.3.0)

* **手書き文字抽出 (OCR) の超高精度化（生成AI導入）:**
    * 抽出エンジンにGoogleの最新生成AI「Gemini」を搭載しました。
    * 手書きの表を文脈で理解し、**「結合セル等の空欄の自動補完」「日付の自動変換」「完璧なExcelセルへの振り分け」**を行い、理想的な状態のExcelファイル（`_抽出後.xlsx`）を生成します。
    * **⚠️ APIキー流出防止の完全対応:** APIキーはプロジェクトフォルダ内には保存されず、PCのユーザーフォルダ内に隠しファイルとして安全に隔離して保存されるため、誤ってGitHubに公開される事故を100%防ぎます。
    * **⚠️ 重要: AI機能を使用するための準備**
      本機能を使用するには、無料のGemini APIキーが必要です。
      1. [Google AI Studio](https://aistudio.google.com/app/apikey) にアクセスし、「Create API key」からキーを取得してください。
      2. アプリ上部のメニューバー「設定」>「Gemini APIキー設定」を開き、取得したキーを貼り付けて保存してください。

## 機能一覧

* **結合:** 複数PDFを1つにマージ
* **分割:** 1ファイルをページごとに分割
* **回転:** 全ページを指定角度で回転
* **Text抽出:** テキストデータを抜き出し
* **手書き抽出:** 生成AIを用いて表組み・空欄・日付を自動補完し、Excelへ出力
* **Excel変換:** 活字の表データをExcelへ
* **画像変換:** JPEG / PNG へ変換
* **DXF変換:** CADデータへ変換 (高精度ハイブリッド版)
