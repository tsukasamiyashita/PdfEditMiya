# PdfEditMiya - PDF Multi-Tool

PdfEditMiyaは、PDFファイルの操作（結合、分割、回転）や、PDFからのデータ抽出（テキスト、Excel、画像、DXF）を一括で行うためのGUIアプリケーションです。
特にDXF変換機能は、CAD由来のベクターデータとスキャン画像（ラスタデータ）の両方に対応し、高精度なCADデータ出力をサポートします。

## 機能一覧

* **結合:** 複数のPDFを1つのファイルに結合
* **分割:** 1つのPDFをページごとに分割
* **回転:** PDFのページを指定角度（90°, 180°, 270°）で回転
* **Text抽出:** PDF内のテキストデータを抽出してtxtファイルに保存
* **Excel変換:** PDF内の表データを解析し、Excelファイルに変換
* **画像変換:** PDFを画像（JPEG / PNG）に変換
* **DXF変換 (高精度):** * CAD由来のPDF: 線分、矩形、曲線を正確に変換
    * スキャンPDF: ノイズ除去と輪郭近似を行い、CADで編集可能なデータに変換
