# PdfEditMiya v1.5.0

PdfEditMiyaは、PDFファイルの操作とデータ変換を一括で行う多機能ツールです。

## 新機能 (v1.5.0)

* **AI抽出ロジックの抜本的見直しと安定化:**
    * 複雑すぎたAIプロンプトを整理し、Markdown表によるレイアウト崩れや白紙出力を完全に防止しました。
    * 手書き文字の分母（/12）が（/2）と誤読される現象に対し、前後の文脈から論理的に連番を推測させる絶対ルールを適用しました。
* **AI文字抽出の高度なカスタマイズUI:**
    * メイン画面に「AI抽出設定」枠を追加しました。メニューを開かなくてもAPIキーの入力やテストが可能です。
    * **エンジン選択:** 最新の生成AIである「Gemini API」か、ローカル動作の「Tesseract」を選択できます。
    * **出力形式選択:** 抽出した文字データの書き出し形式を `.xlsx`、`.csv`、`.txt` の中から自由に選べるようになりました。
    * **⚠️ APIキーの安全保存:** 入力したAPIキーはPCのユーザーフォルダ内に隠しファイルとして隔離・保存されるため、誤ってGitHub等のパブリックな場所にアップロードされる事故を完全に防ぎます。

## 機能一覧

* **結合:** 複数PDFを1つにマージ
* **分割:** 1ファイルをページごとに分割
* **回転:** 全ページを指定角度で回転
* **Text抽出:** PDF内部のテキストデータを抜き出し
* **AI抽出:** 手書き文字やスキャン画像をAI(Gemini/Tesseract)で高精度に読み取り、指定形式へ出力
* **Excel変換:** 活字の表データをExcelへ
* **画像変換:** JPEG / PNG へ変換
* **DXF変換:** CADデータへ変換 (高精度ハイブリッド版)
