# PdfEditMiya v1.10.0

PdfEditMiyaは、PDFファイルの操作とデータ変換を一括で行う多機能ツールです。

## 新機能と改善 (v1.10.0)

* **「データ集約のみ」機能の追加:**
    * UIの実行アクション枠に「データ集約のみ」ボタンを新設しました。
    * これにより、過去に抽出したExcelやCSVファイルなどを後から選択し、1つのマスターデータファイルに結合・集約することが可能になります。
* **出力連番の自動ゼロパディング（スマートソート）:**
    * ファイルの分割機能や画像変換など、1つのPDFから複数ファイルが出力される際、自動的に元のページ総数を計算し、必要な桁数でゼロ埋めを行うように改良しました。（例：全150ページなら `_001.jpg` 〜 `_150.jpg`）
    * エクスプローラー上で名前順に並べた際に桁崩れが起きず、美しいソート順を保持できます。
* **ファイル選択拡張子の柔軟化:**
    * PDFだけでなく、Excel、CSV、テキスト形式のファイルを直接選択して処理（集約など）に流せるように、ファイル選択ダイアログの仕様を拡張しました。

## 機能一覧

* **結合:** 複数PDFを1つにマージ
* **分割:** 1ファイルをページごとに分割
* **回転:** 全ページを指定角度で回転
* **Text抽出:** PDF内部のテキストデータを抜き出し
* **AI抽出:** 手書き文字やスキャン画像をAI(Gemini/Tesseract)で高精度に読み取り、指定形式へ出力
    * カラムの自動認識による不要データの排除機能を搭載
    * 集約データへの「元ファイル名」列の自動付与（トレーサビリティの確保）
    * データ間の列ズレを防止する自動マッピング・補正機能
* **データ集約のみ:** すでに抽出済みのデータ（xlsx/csv/txt）を1つのマスターに合体
* **Excel変換:** 活字の表データをExcelへ
* **画像変換:** JPEG / PNG へ変換
* **DXF変換:** CADデータへ変換 (高精度ハイブリッド版)
