# PdfEditMiya v1.9.0

PdfEditMiyaは、PDFファイルの操作とデータ変換を一括で行う多機能ツールです。

## 新機能と改善 (v1.9.0)

* **データ結合の防止（列の独立性確保）:**
    * 2つの列のデータ（例：「図番」と「品名」など）が誤って1つの列に繋がって出力されてしまう問題を解決しました。
    * AIに対するプロンプトを強化し、「表の縦罫線や、文字列間の大きな空白を列の区切りとして厳密に認識し、本来分かれるべきデータを絶対に1つのキーに結合しないこと」を強力なルールとして強制しています。
* **集約データにおける「列ズレ・行ズレの自動補正機能」:**
    * ページごとにAIの出力結果（キー名や列数）に揺らぎが発生した場合でも、集約データ（マスターファイル）を作成する際に**「インデックス（列の順番）を基準にして、マスターの列数へ強制的に補正・マッピングする」**強固なロジックを実装しています。
    * 個別出力ファイルはAIが読み取ったままの構造を維持しつつ、集約データのみ「足りない列には空文字を埋め、超過した列は整える」という処理を自動で行うため、ファイル間で列がズレる現象が構造上発生しません。

## 機能一覧

* **結合:** 複数PDFを1つにマージ
* **分割:** 1ファイルをページごとに分割
* **回転:** 全ページを指定角度で回転
* **Text抽出:** PDF内部のテキストデータを抜き出し
* **AI抽出:** 手書き文字やスキャン画像をAI(Gemini/Tesseract)で高精度に読み取り、指定形式へ出力
    * カラムの自動認識による不要データの排除機能を搭載
    * 集約データへの「元ファイル名」列の自動付与（トレーサビリティの確保）
    * AI自己分析（Chain of Thought）による空欄認識の強化
* **Excel変換:** 活字の表データをExcelへ
* **画像変換:** JPEG / PNG へ変換
* **DXF変換:** CADデータへ変換 (高精度ハイブリッド版)
