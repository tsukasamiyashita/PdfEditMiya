# PdfEditMiya - 高性能PDFデータ抽出・編集ツール (v2.10.0)

PdfEditMiyaは、PDFファイルから表データやテキストを抽出し、ExcelやCSVなど、使いやすい形式に変換するためのWindows専用ツールです。最新のAIモデル（Gemini API）やOCR技術を搭載しており、手書きの書類や複雑なレイアウトのPDFでも高精度に読み取ることができます。

## 主な機能

### 1. 自由自在なデータ抽出
- **表として抽出**: PDF内の罫線や空白を認識し、エクセルのセルごとに綺麗に分割して出力します。
- **テキストのみ抽出**: 指定した行（水平線）や列（垂直線・縦書き用）のテキストをピンポイントで抜き出します。
- **複数範囲の同時指定**: 1ページ内の複数箇所を一度に選択し、それらを横並びに結合して出力することも可能です。

### 2. 強力な3つの抽出エンジン
- **標準ライブラリ**: デジタル作成されたPDFから高速・正確にデータを抽出します。
- **Gemini API (AI)**: Googleの最新AIを活用。手書き文字、かすれた文字、複雑な表構造も人間のように理解して読み取ります。
- **Tesseract (ローカルOCR)**: ネット環境がない場所でも、お使いのPC内で文字認識処理を行えます。

### 3. 多彩な出力フォーマット
- **データ系**: Excel (.xlsx), CSV, JSON, Markdown, Word (.docx)
- **画像・図面系**: JPEG, PNG, SVG, TIFF, BMP, DXF (CAD用変換)
- **テキスト系**: プレーンテキスト (.txt)

### 4. 便利なPDF編集・管理ツール
- **結合・分割・回転**: 複数のPDFを1つにまとめたり、ページごとにバラバラにしたり、向きを直したりできます。
- **スマートデータ集約**: 複数ファイルの表データを、列名を自動で判別して1つの大きなリストにまとめます。「〃」などの省略記号も自動で上のセルの内容を補完します。
- **データ単純結合**: 加工せず、シンプルにファイルを縦に繋ぎ合わせることも可能です。
- **PDFデータ構造を確認**: 読み込んだPDFが「文字データ」か「画像（スキャン）」かを自動判定し、最適なモードをアドバイスします。

## 使い方（基本の流れ）

1. **ファイルを読み込む**: アプリ画面の「ファイルを選択」または「フォルダを選択」ボタンを押して対象を選びます。
2. **モードを選ぶ**: 「表として抽出」または「テキストのみ抽出」を選択します。
3. **範囲を決める**: 「抽出範囲を選択」ボタンを押し、マウスで読み取りたい場所を囲みます。
4. **実行する**: 「実行（保存先を選択）」ボタンを押し、名前を付けて保存すれば完了です！

## 動作環境
- **OS**: Windows 10 / 11 (64bit)
- **その他**: AIエンジン（Gemini）を使用する場合はインターネット接続が必要です。

## インストール・実行方法

1. ダウンロードした `PdfEditMiya-v2.10.0.zip` を右クリックして「すべて展開」します。
2. 展開されたフォルダ内にある `PdfEditMiya-v2.10.0.exe` をダブルクリックして起動します。

---
### ⚠️ 起動時に警告画面が表示される場合
本ソフトは開発元が未登録のため、初回起動時にWindows Defender (SmartScreen) の警告が表示されることがあります。以下の手順で安全に実行できます。

1. 青い画面（WindowsによってPCが保護されました）が表示されたら、左上の **「詳細情報」** をクリックします。
2. 右下に表示される **「実行」** ボタンをクリックすると、アプリが起動します。
