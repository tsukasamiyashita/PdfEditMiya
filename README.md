# PdfEditMiya v1.10.0

PdfEditMiyaは、PDFファイルの操作とデータ変換を一括で行う多機能ツールです。

## 新機能と改善 (v1.10.0 Update 9)

* **空白セルのインテリジェント自動補完（ファイル境界越え対応）:**
    * 集約データを作成する際、セルの下の行が「空白」または「繰り返し記号（〃など）」になっていた場合に、自動的に「上の行のテキスト」を引き継いで空白を埋める機能を搭載しています。
    * 今回の改善で、**別々のファイル（ページ）に分かれて抽出されたデータを集約する際にも、引継ぎがリセットされず、シームレスに前のファイルのテキストを引き継ぐ**ようになりました。
    * 依然として、「備考」列や、数値・記号のみのデータは引き継がずに空白を維持する厳格なルールは機能しています。
* **「1列へのデータ結合」問題の完全な排除 (バグ修正):**
    * AIが勝手に「タプル形式の文字列 `"(値1, 値2)"`」などで出力してしまい、そのまま1つのセルにすべて合わさってしまうというパースエラーを修正しました。プログラム側で文字列を解析し、強引にリストへ展開・分割する強力な安全装置を組み込んでいます。
* **数値整合性による「スマート・カラムマッピング」の究極進化:**
    * カラム（ヘッダー名）のテキストに依存して列を判別する処理を完全に廃止しました。集約データを作成する際、各列に含まれる「数値の割合」や「分数の割合」、「桁数」のみを計算・比較し、**数値の整合性だけで適正な列へ自動判別・マッピングさせる**アルゴリズムへと刷新しました。（テキストデータ同士の場合は、元の列の順番を優先して配置します）

## 機能一覧

* **結合:** 複数PDFを1つにマージ
* **分割:** 1ファイルをページごとに分割
* **回転:** 全ページを指定角度で回転
* **Text抽出:** PDF内部のテキストデータを抜き出し
* **Excel変換:** 活字の表データをExcelへ
* **画像変換:** JPEG / PNG へ変換
* **DXF変換:** CADデータへ変換 (高精度ハイブリッド版)
* **AIデータ抽出:** 手書き文字やスキャン画像をAI(Gemini/Tesseract)で高精度に読み取り、個別ファイルに出力
* **データ集約:** すでに抽出済みのデータ（xlsx/csv/txt）を1つのマスターに合体（列幅自動調整・空白行自動削除つき）
