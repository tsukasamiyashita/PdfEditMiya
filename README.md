PdfEditMiya v2.0.0

PdfEditMiyaは、PDFファイルの編集（結合・分割・回転）から、AIや標準ライブラリを用いた高度なデータ抽出、そして抽出したデータの自動集約までをワンストップで行うWindows向けデスクトップアプリケーションです。

🌟 主な特徴

直感的な操作: ファイルやフォルダを選択し、処理を選んで実行するだけの簡単設計。

高精度AIデータ抽出: Google Gemini APIを利用し、かすれた手書き文字や複雑な表の罫線を高精度に認識。

ローカルOCR対応: オフライン環境でも使えるTesseract OCRに対応。

多彩な出力フォーマット: Excel (.xlsx)、CSV、Text、JSON、Markdown、Word (.docx)、DXF、各種画像フォーマット、SVGベクター出力に対応。

ピンポイント抽出（クロップ機能）: プレビュー画面からドラッグで指定した範囲（複数可）のみを抽出可能。

スマートデータ集約: バラバラに抽出された複数ページの表データを、列名を自動認識して1つのマスターファイルに結合・集約（AIによる高度な意味的統合にも対応）。

設定の保存機能: APIキーや出力形式など、現在の選択状態を保存し次回起動時に復元可能。

🚀 基本的な使い方

アプリの起動: PdfEditMiya-v2.0.0.exe をダブルクリックして起動します。

対象の選択: 画面上部の「📄 ファイルを選択」または「📁 フォルダを選択」から処理したいPDFを選びます。

設定の調整:

保存先: 同じフォルダ、または任意のフォルダを指定。

抽出設定: 使用するエンジン（標準 / Tesseract / Gemini）と出力形式を選択。

範囲指定（任意）: 特定の表や箇所だけを抽出したい場合は「抽出範囲を選択」から領域を指定します。

実行: 画面下部のボタン（「🚀 選択した抽出・変換を実行」や「結合」など）を押して処理を開始します。

⚙️ AI機能を使うための事前準備

本アプリの強力な「AIデータ抽出」機能をフル活用するためには、お使いのPCで以下の簡単な準備が必要です。

1. Gemini API を使う場合（推奨・超高精度）

最新のAIモデルを利用して高精度なテキスト・表データの抽出を行います。インターネット接続が必要です。

Google AI Studio にアクセスし、お手持ちのGoogleアカウントでログインします。

画面左上の「Create API key」を押し、新しいプロジェクトでAPIキーを作成します。

発行された長い文字列（APIキー）をコピーし、本アプリの設定画面内「[AI用] APIキー」の欄に右クリックで貼り付けます。

「テスト」ボタンを押し、通信が成功することを確認してください。
※入力したキーは「💾 現在の選択項目を保存」ボタンを押すことで保存され、次回以降入力不要になります。

2. Tesseract を使う場合（オフライン・簡易抽出）

インターネットに繋がっていない環境でも使用できる無料のOCRエンジンです。

Tesseractインストーラー配布ページ にアクセスします。

最新の64bit版インストーラー（tesseract-ocr-w64-setup...exe）をダウンロードして実行します。

インストール中の「Choose Components」画面で、Additional language data (download) の中にある Japanese と Japanese (vertical) に必ずチェックを入れてください。

インストール先は初期設定のまま（C:\Program Files\Tesseract-OCR）変更せずに完了させてください。

⚠️ 注意事項・トラブルシューティング

初回起動時の遅延: アプリケーション（exeファイル）の初回起動時や、PC起動直後の最初の実行時は、内部ファイルの展開処理が行われるため、画面が表示されるまでに十数秒〜数十秒ほど時間がかかる場合があります。そのままお待ちください。

セキュリティソフトの警告: 本アプリはプログラムを単一の実行ファイル（exe）にパッケージ化している性質上、Windows Defenderなどのセキュリティソフトによって「認識されないアプリ」として警告が出たり、起動がブロックされたりする場合があります。その場合は「詳細情報」をクリックしてから「実行」を選択してください。

処理の中断: 大量のファイルを処理中に「処理を中止」ボタンを押した場合、現在実行中のファイル処理が終わったタイミングで安全に停止します。即座に停止するわけではありません。

Word (.docx) 出力について: AIエンジン等で Word 形式の出力を行った場合、表のレイアウトが崩れることがあります。複雑な表データの抽出には Excel (.xlsx) 出力を強く推奨します。
