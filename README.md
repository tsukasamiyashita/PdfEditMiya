# PdfEditMiya v1.2.0

PdfEditMiyaは、PDFファイルの操作とデータ変換を一括で行う多機能ツールです。
本バージョン (v1.2.0) では、「DXF変換」の解析エンジンを刷新し、実用性を大幅に向上させました。

## 新機能 (v1.2.0)

* **DXF変換の高精度化:**
    * **ベクター処理:** 曲線（ベジェ曲線）や円弧を滑らかなスプラインとして出力可能になりました。
    * **ラスタ処理:** スキャンされた図面PDFに対し、ノイズ除去と輪郭近似を行い、クリーンな線画データを生成します。

## 機能一覧

* **結合:** 複数PDFを1つにマージ
* **分割:** 1ファイルをページごとに分割
* **回転:** 全ページを指定角度で回転
* **Text抽出:** テキストデータを抜き出し
* **Excel変換:** 表データをExcelへ
* **画像変換:** JPEG / PNG へ変換
* **DXF変換:** CADデータへ変換 (高精度ハイブリッド版)
