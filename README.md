# PdfEditMiya v1.5.0

PdfEditMiyaは、PDFファイルの操作とデータ変換を一括で行う多機能ツールです。

## 新機能と改善 (v1.5.0)

* **AI文字抽出の抜本的見直しと安定化:**
    * **列ズレの完全防止**: AIからの出力を構造化されたJSONスキーマに強制させることで、テキストデータのカンマ混入に起因する列ズレ問題をアーキテクチャレベルで排除しました。
    * **入念な画像前処理**: 画像をAIへ渡す前に、「ノイズ除去」→「局所的コントラスト強調（CLAHE）」→「モルフォロジー演算（かすれた文字の物理的な太字化・結合）」の強力な多段パイプラインを自動でかけ、手書き文字の抽出精度を飛躍的に向上させています。
    * **文脈推論の強化**: 「1/12」が「1/2」になる桁落ちや、「ケッチ台車」が「写真事業」になる誤変換など、具体的なエラー事例を教訓としてAIのプロンプトに組み込み、論理的で文脈に沿った補完を行うよう厳密に指示しています。

## 機能一覧

* **結合:** 複数PDFを1つにマージ
* **分割:** 1ファイルをページごとに分割
* **回転:** 全ページを指定角度で回転
* **Text抽出:** PDF内部のテキストデータを抜き出し
* **AI抽出:** 手書き文字やスキャン画像をAI(Gemini/Tesseract)で高精度に読み取り、指定形式へ出力
* **Excel変換:** 活字の表データをExcelへ
* **画像変換:** JPEG / PNG へ変換
* **DXF変換:** CADデータへ変換 (高精度ハイブリッド版)
