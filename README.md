PdfEditMiya v2.6.0

PdfEditMiyaは、PDFファイルの編集（結合・分割・回転）から、AIを用いた高度なデータ抽出、そして抽出したデータの自動集約までをワンストップで行うWindows向けデスクトップアプリケーションです。

🌟 主な特徴

直感的な操作: ファイルやフォルダを選択し、処理を選んで実行するだけの簡単設計。

高精度AIデータ抽出: Google Gemini APIを利用し、かすれた手書き文字や複雑な表の罫線を高精度に認識。無料枠でもバーストエラーを自動回避する機能を搭載。

柔軟な「抽出モード」: 用途に合わせて以下の2つのモードを切り替えられます。

表として抽出: 罫線や空白を認識し、エクセルのセルごとに分割して出力します。

テキストのみ抽出: 表構造を無視し、読み取った文章を改行ごとに分割して、横一列（1行）にズラッと展開して出力します。

スマートデータ集約: バラバラに抽出された複数ページの表データを、列名を自動認識して1つのマスターファイルに結合・集約。

【強化】同上コピー機能: 集約時に「〃」「同上」「...」などの省略記号があった場合、自動的に「上のセルの内容」をコピーしてデータを補完します（空白のセルはそのまま空白として正しく維持されます）。

PDF内部構造の自動判定: 標準ライブラリ（テキスト抽出）を実行した際、対象が「スキャンされた画像（ラスターデータ）」であるかを事前に検知し、無駄な処理を行う前にAIエンジンの使用を提案します。また、「PDFデータ構造を確認」ボタンから手動で最適なエンジンを調べることも可能です。

高度なAPI詳細設定: 「無料枠」と「課金枠」それぞれで独立して設定・保存できます。

使用モデル: 目的に合わせた Gemini モデルの選択（ボタン一つで最新のモデルリストをAPIから自動取得できます）。

RPM制限 / スレッド数: バーストエラーを防ぐ制限設定や、超高速な「並列処理」の調整。

カスタムプロンプト: 複数行にわたる独自のルール（金額のカンマ削除、日付形式など）を指示でき、ワンクリックでお気に入りとして保存・復元が可能です。

ローカルOCR対応: インターネットに繋がらないオフライン環境でも使える「Tesseract OCR」に対応。

多彩な出力フォーマット: Excel (.xlsx), CSV, テキスト, JSON, Markdown, Word (.docx)、各種画像フォーマット出力に対応。

ピンポイント抽出（クロップ機能）: プレビュー画面からドラッグで指定した範囲（複数可）のみを超高速で抽出可能。指定した枠に一部でも接している文字も逃さず抽出します。

設定のクリーンな保存: APIキーや出力形式などの設定は、ご自身のユーザーフォルダ内の PdfEditMiya フォルダに自動で整理して保存され、次回起動時に復元されます。

🚀 基本的な使い方

アプリの起動: ダウンロードした PdfEditMiya-v2.6.0.exe をダブルクリックして起動します。

対象の選択: 画面上部の「📄 ファイルを選択」または「📁 フォルダを選択」から処理したいPDFやデータを選びます。

設定の調整:

保存先: 同じフォルダ、または任意のフォルダを指定。

データ抽出・変換設定: 使用するエンジン（標準 / Tesseract / Gemini）と出力形式、抽出モードを選択。

範囲指定（任意）: 特定の表や箇所だけを抽出したい場合は「抽出範囲を選択」から領域を指定します。

実行: 画面下部のボタン（「🚀 選択した抽出・変換を実行」や「結合」など）を押して処理を開始します。

⚠️ 【重要】初回起動時の注意点（青い警告画面について）

本アプリ（.exeファイル）を初めて実行した際、青い画面で**「Windows によって PC が保護されました」**という警告（SmartScreen）が表示される場合があります。
これはインターネットからダウンロードした新しいアプリに対して表示されるWindowsの標準的な保護機能であり、ウイルスではありませんのでご安心ください。

【回避して起動する手順】

警告画面の左側にある 「詳細情報」 という文字をクリックします。

右下に 「実行」 ボタンが新しく表示されるので、それをクリックするとアプリが立ち上がります。

※ 初回起動時は、裏側でアプリを動かすための準備処理が行われるため、画面が表示されるまでに十数秒〜数十秒ほど時間がかかる場合があります。ボタンを押した後はそのままお待ちください。

⚙️ AI機能を使うための事前準備

本アプリの強力な「AIデータ抽出」機能をフル活用するためには、以下の簡単な準備が必要です。

🤖 Gemini API を使う場合（推奨・超高精度）

最新のAIモデルを利用して高精度なテキスト・表データの抽出を行います。インターネット接続が必要です。

ブラウザで Google AI Studio にアクセスし、Googleアカウントでログインします。

画面左上の「Create API key」を押し、新しいプロジェクトでAPIキーを作成します。

発行された長い文字列（APIキー）をコピーし、本アプリのメイン画面から「⚙️ 詳細設定 (APIキー / モデル / 制限)」ボタンを押します。

設定したいプラン（無料枠 または 課金枠）のタブを開き、その中の「APIキー」欄に右クリックで貼り付けます。

「テスト」ボタンを押し、通信が成功することを確認してください。

※使用するモデルや、その他の高度な抽出設定も、各タブごとに別々に設定できます。

画面下部の**「設定を適用して閉じる」**ボタンを押して設定を完了します。

最後に、メイン画面上部の「実行プランの選択」で、実際にデータ抽出で使用するプランを選んでください。

※入力した設定は、メイン画面の「💾 現在の選択項目を保存」ボタンを押すことで PdfEditMiya フォルダに保存され、次回から入力不要になります。

📝 Tesseract（ローカルOCR）を使う場合のご準備（オフライン環境向け）

インターネットに接続できない環境でも、無料の文字認識（OCR）機能をご利用いただけます。ご利用には、お使いのパソコンに「Tesseract（テッセラクト）」という専用ソフトを追加していただく必要がございます。

ダウンロード: Tesseractインストーラー配布ページ を開きます。

ページ内にある「tesseract-ocr-w64-setup...exe」のリンクをクリックし、ファイルをパソコンにダウンロードします。

インストール: ダウンロードしたファイルをダブルクリックして開き、「Next」で進めます。

【重要】日本語データの追加: 途中で表示される「Choose Components」画面で、Additional language data (download) の左側の「+」マークを展開し、「Japanese」 と 「Japanese (vertical)」 の2箇所に必ずチェックを入れてください。

完了: 「Destination Folder（インストール先）」の画面では、絶対に場所を変更せず（初期設定の C:\Program Files\Tesseract-OCR のまま）、「Next」を押して完了させてください。
