# PdfEditMiya v1.6.0

PdfEditMiyaは、PDFファイルの操作とデータ変換を一括で行う多機能ツールです。

## 新機能と改善 (v1.6.0)

* **カラム（ヘッダー）の自動認識と集約データの高度化:**
    * AIが画像の表から「カラム名（ヘッダー）」と「データ行」を自動的に分離・認識する機能を搭載しました。
    * これにより、表のタイトルや日付、ページ番号など、データ集約において**邪魔になる不要な文字列を自動で完全に無視（除外）**することが可能になりました。
    * 集約マスターデータを作成する際、一番上にカラム名を1行だけ配置し、その下には全ファイル・全ページの純粋なデータ行のみが縦に連結されるため、人間が後から成形し直す手間が大幅に削減されます。
* **AI文字抽出の高精度化（前バージョンからの継承）:**
    * AI出力をJSONスキーマ化することで、カンマ混入等による列ズレをシステムレベルで排除しています。
    * ノイズ除去やモルフォロジー演算（かすれた文字の太字化）によるOpenCV画像前処理を自動で行い、認識精度を向上させています。

## 機能一覧

* **結合:** 複数PDFを1つにマージ
* **分割:** 1ファイルをページごとに分割
* **回転:** 全ページを指定角度で回転
* **Text抽出:** PDF内部のテキストデータを抜き出し
* **AI抽出:** 手書き文字やスキャン画像をAI(Gemini/Tesseract)で高精度に読み取り、指定形式へ出力
* **Excel変換:** 活字の表データをExcelへ
* **画像変換:** JPEG / PNG へ変換
* **DXF変換:** CADデータへ変換 (高精度ハイブリッド版)
