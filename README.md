# PdfEditMiya v1.11.0

PdfEditMiyaは、PDFファイルの操作とデータ変換を一括で行う多機能ツールです。

## 新機能と改善 (v1.11.0 Update)

* **通信の安定化と確実な処理 (NEW):**
    * AIデータ抽出（Gemini API）で複数の範囲を選択した場合、API側の利用制限によるリトライ遅延を回避するため、内部で確実な直列処理を行うように設計を差し戻し、最も安定する形に最適化しました。
* **APIレートリミット(429)対策 (NEW):**
    * Gemini APIの無料枠制限（通常15RPM程度）に引っかからないよう、各領域・ページの抽出ごとに約4秒間の自動クールダウン処理を挟むようにしました。
    * 再試行ロジックにJitter（揺らぎ）を加え、大量のページを連続処理する際の堅牢性が飛躍的に向上しています。
* **抽出範囲の「全体に戻す」リセットボタン (NEW):**
    * 一度範囲を選択した後でも、やっぱりページ全体をAIに読み取らせたい時にワンクリックで設定をリセットできる専用ボタンを追加しました。

[ 過去の主要なアップデート ]
* **ページ番号の形式変更:** AI抽出データのA列に「現在のページ/総ページ数（例: 1/5）」を自動付与。
* **縦書き文字の自動結合機能:** 縦書きテキストが1文字ずつバラバラになる問題を解消し、自動的に横文字に結合して出力。
* **抽出範囲の「1列化」と個別ファイル化:** 範囲選択時は1つの選択領域を「1つの列」にまとめて出力。
* **抽出範囲プレビューの拡大機能:** プレビュー画面に「拡大・縮小・フィット」ボタンとスクロールバーを実装。
* **データ集約時のソート改善:** データを1つにまとめる際、ファイルの順番が「更新日時順」になるよう改善。

## 機能一覧

* **結合:** 複数PDFを1つにマージ
* **分割:** 1ファイルをページごとに分割
* **回転:** 全ページを指定角度で回転
* **Text抽出:** PDF内部のテキストデータを抜き出し
* **Excel変換:** 活字の表データをExcelへ
* **画像変換:** JPEG / PNG へ変換
* **DXF変換:** CADデータへ変換 (高精度ハイブリッド版)
* **AIデータ抽出:** 手書き文字やスキャン画像をAI(Gemini/Tesseract)で高精度に読み取り、個別ファイルに出力
* **データ集約:** すでに抽出済みのデータ（xlsx/csv/txt）を1つのマスターに合体（列幅自動調整つき）
