# PdfEditMiya v1.10.0

PdfEditMiyaは、PDFファイルの操作とデータ変換を一括で行う多機能ツールです。

## 新機能と改善 (v1.10.0 Update 11)

* **集約データ出力ファイル名の固定化:**
    * 今まで親フォルダ名などに影響されて変動していた出力ファイル名を、いかなる場合もシンプルに「データ集約.xlsx（.csv / .txt）」として保存するように修正しました。
* **集約時の不要な空白行の自動削除:**
    * 集約データを作成する際、元ファイル名以外のセルがすべて空白（空文字やNone）になっている不要な行を自動で検知し、ファイルから完全に排除する機能を実装しました。これにより、よりクリーンなデータベースが構築されます。
* **「1列へのデータ結合」問題の完全な排除:**
    * 抽出データが集約される際、Excelから読み込んだ1行分のデータ（Pythonのタプル型）が「丸ごと1つの文字列」に誤変換され、すべて1つのセルに入ってしまうという型判定のバグを修正しました。
* **数値整合性による「スマート・カラムマッピング」の究極進化:**
    * カラム（ヘッダー名）のテキストに依存して列を判別する処理を完全に廃止しました。
    * 集約データを作成する際、各列に含まれる「数値（数字を含むデータ）の割合」や「分数の割合」、「桁数」のみを計算・比較し、数値の整合性だけで適正な列へ自動判別・マッピングさせるアルゴリズムへと刷新しました。（テキストデータ同士の場合は、元の列の順番を優先して配置します）

## 機能一覧

* **結合:** 複数PDFを1つにマージ
* **分割:** 1ファイルをページごとに分割
* **回転:** 全ページを指定角度で回転
* **Text抽出:** PDF内部のテキストデータを抜き出し
* **Excel変換:** 活字の表データをExcelへ
* **画像変換:** JPEG / PNG へ変換
* **DXF変換:** CADデータへ変換 (高精度ハイブリッド版)
* **AIデータ抽出:** 手書き文字やスキャン画像をAI(Gemini/Tesseract)で高精度に読み取り、個別ファイルに出力
* **データ集約:** すでに抽出済みのデータ（xlsx/csv/txt）を1つのマスターに合体（列幅自動調整つき）
